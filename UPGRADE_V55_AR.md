# تحديث المنصة V55

## أهم التغييرات

- تصميم هادئ وبسيط ومتجاوب مع الهاتف دون حذف أي خدمة موجودة.
- استبدال صورة المدرس بالصورة الجديدة مع عرض الجزء العلوي فقط.
- الحجز يصدر فورًا ثلاثة أكواد رقمية فقط: الطالب، ولي الأمر، ومتابعة الحجز.
- بوابة الطالب وولي الأمر تعملان فور الحجز مع توضيح أن القبول ما زال معلقًا.
- قبول الحجز يستخدم نفس الأكواد، ورفضه يوقف الحساب المؤقت بأمان.
- قسم مستقل وسهل لإدارة المواعيد والمجموعات داخل لوحة المدرس.
- إشعارات فورية بالحجوزات داخل لوحة المدرس، وتجهيز كامل لإشعارات الهاتف عند إغلاقها.
- إصلاح توافق إنشاء واستعادة النسخ الاحتياطية V54.

## تحسينات V55.1

- نموذج الحجز أصبح خطوات قصيرة مثل نموذج المعاينة.
- منشئ الامتحان أصبح خانات: سؤال، أربع إجابات، واختيار الإجابة الصحيحة.
- موعد الحصة أصبح وقتًا واحدًا فقط بدون وقت نهاية.
- إعادة تصميم الحضور مع قائمة الطلاب وأزرار حاضر وغائب وQR.
- إعادة تصميم الدفع والبحث فيه، وضغط بطاقات التقييمات.
- إزالة الإعدادات والنسخ الاحتياطي وأخطاء النظام من واجهة المدرس لتقليل الزحام، مع بقاء وظائف الحماية في الخلفية.

## تفعيل إشعارات الهاتف عند إغلاق لوحة المدرس

1. افتح Firebase Console ثم Project settings ثم Cloud Messaging.
2. من Web Push certificates أنشئ مفتاحًا إن لم يكن موجودًا.
3. انسخ المفتاح العام فقط إلى `messagingVapidKey` داخل `assets/firebase-config.js`.
4. انشر Functions وHosting، ثم افتح لوحة المدرس واضغط «تنبيهات الحجز» واسمح بالإشعارات.

بدون المفتاح تعمل التنبيهات فورًا أثناء فتح لوحة المدرس، لكن الإشعار أثناء إغلاق التطبيق يحتاج مفتاح Web Push العام.
